Add custom color theme

First, add your CSS theme color code before you link the CSS-Framework. Secondly add "addthemecolor('yourcolorthemename');" in your js code. (CSS color theme template below)

body:not(.style-scope)[yourcolorthemename] {
--Background: #181818;
--Primary: 187, 134, 252;
--Secondary: 3, 218, 198;
--OnBackground: 255, 255, 255;
--OnPrimary: 0, 0, 0;
--OnSecondary: 0, 0, 0;
--Popup: #FFFFFF;
--PopupLabel: 0, 0, 0;
--sliderthumbcolor: var(--Primary);
--hover: brightness(80%);
}


addthemecolor('yourcolorthemename');
